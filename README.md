# High-Frequency-Devices-Second-Series
This work is about the calculation of dielectric constant, the analysis of an antenna array, usage of stubs and resonances. The final report PDF is included in parts due to file size problems alongside the assignment prompt and the MATLAB codes. 
This work was assigned for the course of High Frequency Devices at the Aristotle University of Thessaloniki by professor Traianos Gioultsis.
